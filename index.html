Some of rule sets

The best practice to set border-box to box-sizing into html selector:

html {box-sizing: border-box}

The best practice to clear margin and padding from all the page using * selector:

* {margin: 0; padding: 0}

The best practice is to import new fonts via import them into your CSS file using @import:

@import url('font source url')
